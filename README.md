## About the Project

**App Financeiro** is a clean, production-deployed personal finance management SPA. Built with a component-driven React architecture, deployed to GitHub Pages, and maintained as a live project.

The application focuses on **clarity over feature bloat** — a minimal, fast interface for tracking personal finances without overwhelming the user.

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | React · Vite |
| Language | JavaScript |
| Architecture | Single Page Application (SPA) |
| State | React Hooks |
| Deployment | GitHub Pages |

---

## Engineering Highlights

- **Component-driven architecture** with clear separation of concerns
- **Stateful UI** managed via React Hooks (no external state library needed)
- **Production deployment** via GitHub Pages CI
- **Fast build pipeline** powered by Vite
- **Responsive layout** working across mobile and desktop

---

## Local Setup

```bash
# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build

# Preview the production build
npm run preview
```

No environment variables required for local development.
